# mealapp

Single Page Aplication

## Tech

- HTML 5
- Tailwind CSS
- Javascript
- Jquery
- Node Js
- GSAP 

## How to run

### Requirements
- Node JS

### Steps
#### 1. Install Dependency

type `npm install` on terminal

#### 2. Run Server

type `npm run start` on terminal

#### 3. Open the URL

open url in the browser `http://localhost:3000`
